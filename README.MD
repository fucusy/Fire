# 业火五笔输入法

> 使用 `swift` 实现的五笔输入法


![](https://github.com/qwertyyb/FireWubi/workflows/CI/badge.svg)

### Install

将编译好的App放到目录/Library/Input Methods，并在系统内设置此输入法

### Usage
中文模式下输入
- zdt 为当前日期，如2020-05-19
